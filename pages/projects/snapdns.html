SnapDNS

The modern, cross-platform DNS switcher.
Enable DNS-over-HTTPS (DoH) on any OS version without the headache.

* Installer handles background service automatically.
Looking for Linux or macOS versions?

Architecture

Split-Process Design: UI runs as User, Background Service runs as System. Uses Named Pipes for secure communication and a local UDP proxy for DoH.

Secure

Enforce encrypted DNS queries (DoH) on any OS version. Your traffic stays private.

Beautiful

Clean Avalonia UI with full Dark & Light theme support matching your system.

Profiles

Comes with Cloudflare, Google, and Quad9 presets. Easily add your own custom servers.

Utilities

Includes handy tools like Flush DNS, Latency Check, and Leak Test built-in.

Open

No tracking, fully open-source (GPLv3). Inspect the code yourself.
